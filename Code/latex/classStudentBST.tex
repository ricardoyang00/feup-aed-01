\doxysubsection{Detailed Description}
A class to represent the Binary search tree to store \textquotesingle{}\mbox{\hyperlink{classStudent}{Student}}\textquotesingle{} objects. 

This class represents a binary search tree for storing and managing student records. Each node in the tree holds a \textquotesingle{}\mbox{\hyperlink{classStudent}{Student}}\textquotesingle{} object.
