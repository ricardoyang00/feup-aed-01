\doxysubsection{Detailed Description}
A structure representing a binary tree node containing \mbox{\hyperlink{classStudent}{Student}} data. 

This structure defines a node for a binary tree, with each node holding a \mbox{\hyperlink{classStudent}{Student}} object and pointers to its left and right children.
