Main \mbox{\hyperlink{classScript}{Script}} for interacting with the \char`\"{}\+LEIC Schedules\char`\"{} system. 

The {\ttfamily \mbox{\hyperlink{classScript}{Script}}} class provides a menu-\/driven interface for users and administrators to manage academic data, process change requests, and access consultation features.
